# Library Management System
This Library Management System is a Java-based web application developed using Servlets, JDBC, and Oracle Database.
The application allows users to manage library books through a simple web interface deployed on Apache Tomcat. 
It demonstrates database connectivity, use of DAO and service layers, and basic CRUD operations with Oracle tables.
This project is mainly developed for academic learning and practice of Java web technologies.

Key Features:

=>Add new books to the library

=>View book details using ISBN

=>Author and book data stored in Oracle Database

=>JDBC used for database connectivity

=>Deployed and tested on Apache Tomcat

Technologies Used:
Java,
Oracle XE,
JDBC,
Apache Tomcat,
HTML.
